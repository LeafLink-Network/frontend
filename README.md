# LearnLeaf Frontend

LearnLeaf는 개발자를 위한 채용 정보 및 커뮤니티 플랫폼입니다.

## 주요 기능

- 채용공고 검색 및 필터링
- 커뮤니티 게시판
- 사이드 프로젝트 모집
- 채용 리포트 및 분석
- 개인화된 사용자 경험

## 기술 스택

- React
- React Router
- Recharts (데이터 시각화)
- React Icons

## 개발 환경 설정

\`\`\`bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start
\`\`\`

## 폴더 구조

- src/components: 재사용 가능한 UI 컴포넌트
- src/pages: 페이지 컴포넌트
- src/assets: 이미지, 아이콘 등의 정적 파일
